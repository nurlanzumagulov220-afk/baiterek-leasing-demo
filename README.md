# Платформа ЕППБ — No-code конструктор с суверенным AI-слоем

**Хакатон Байтерек 2026 · Demo Day 15 мая · Astana Hub**

> No-code конструктор лизинговых услуг с встроенным модулем криптографической верификации и суверенным AI-модулем Aksakal.

---

## Что мы сделали

Классические порталы лизинга — это монолит: каждое новое условие требует разработчика, деплоя, простоя. Мы сделали иначе.

**Genesis Atom** — JSON-файл описывает продукт полностью: ставку, лимиты, поля формы, валидацию, расчётные формулы. Изменить ставку с 18% на 14% — одна строка в файле, эффект мгновенный для всех пользователей. Никакого кода.

**Слой верификации** — встроенный модуль проверки, который работает в три момента:
1. `stamp_atom` — криптографический отпечаток продукта при создании
2. `verify_tol` — проверка заявки до записи в БД: подлинность атома, соответствие лимитам, полнота полей
3. `settle_amanat` — криптографическая подпись утверждённого графика платежей

**AI-модуль (Aksakal)** — локальный Go-сервис, который работает внутри контура. Выполняет ML-скоринг заявок по 4 признакам и поддерживает федеративное обучение без передачи сырых данных вовне. Это полностью соответствует п. 5 ТЗ.

## Быстрый старт

### 1. Aksakal (Go-ядро)
```bash
cd aksakal
go run aksakal.go
# → :8080
```

### 2. Бэкенд
```bash
cd backend
python3 -m venv .venv && .venv/bin/pip install -r requirements.txt
.venv/bin/python create_admin.py          # создать администратора
.venv/bin/uvicorn main:app --reload --port 8000
# → http://localhost:8000/docs
```

### 3. Фронтенд
```bash
cd frontend
npm install && npm run dev
# → http://localhost:5173
```

## Архитектура

```
Предприниматель
      │
      ▼
  React Frontend (5173)
      │
      ▼
  FastAPI Backend (8000)
      ├── Модуль верификации (verification_layer.py)
      │       ├── stamp_atom()    → Штамп на атом
      │       ├── verify_tol()    → Консенсус 3/3 + AI ML-скоринг
      │       └── settle_amanat() → Криптографическая подпись графика
      │
      └── AI-модуль (Aksakal) (8080)
              ├── POST /internal/score    → ML trust_score
              ├── POST /internal/sign     → Криптографическая подпись
              ├── POST /federated/update  → Федеративное обучение
              └── POST /ask               → QA по метрикам заявок
```

| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/atoms` | Список всех продуктов |

| POST | `/admin/ask-aksakal` | Спросить AI-модуль о метриках |
| GET | `/docs` | Swagger UI |

## 0% Hardcode — доказательство

Измените `rate_annual_pct` в `backend/atoms/leasing_wagons_s1.json` с `18.0` на `10.0`:

```json
"params": {
  "rate_annual_pct": 10.0,
  ...
}
```

Перезагрузите страницу калькулятора — ставка изменилась для всех пользователей. Никакого деплоя, никакого кода.
